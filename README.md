# FuckViaCustomTab
**去他妈的Via应用内浏览器（自定义标签）**

</div>

# 作用
Via在版本6.9.0适配了链接应用内打开 导致想要使用浏览器内置的下载管理器等页面需要再点击一个按钮跳转到Via 非常繁琐 本模块用于移除此特性
